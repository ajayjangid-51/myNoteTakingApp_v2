# My Note Taking App

A modern React-based note-taking web application with rich text editing, drawing capabilities, media embedding, and code execution.

## Features

- **One Long Scrollable Page**: All notes are displayed in a single scrollable interface
- **Rich Text Editing**: Bold, italic, underline, headings, lists, checkboxes, and more
- **Inline Screenshots**: Paste images directly into notes
- **Drawing Area**: Add drawable canvases for diagrams and sketches
- **Media Embedding**: Insert and play videos, audio files, and links inline
- **Code Execution**: Write and run JavaScript code directly in notes
- **Dark Mode**: Toggle between light and dark themes
- **Note Management**: Create, rename, and delete notes with a sidebar navigation

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Building for Production

```bash
npm run build
```

## Technologies Used

- React 18
- TypeScript
- Vite
- Slate.js (Rich Text Editor)
- Monaco Editor (Code Editor)
- React Sketch Canvas (Drawing)
- Lucide React (Icons)
- Local Storage (Data Persistence)
